# Prova D3SET
---

### Backend
Após instalar todas as dependências, para iniciar o backend rodar o comando:

```
npm run dev
or 
yarn dev
```

Com isso o servidor será inicializado na porta 3000. Para alterar a porta, edite no arquino src/index.ts.

##### Tecnologias utilizadas:
 * Node
 * Express
 * Typeorm
 * Sqlite
 * Typescript
